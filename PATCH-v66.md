# HYPER-HOST v66

Точечный LAN-only FTP patch:

- ProFTPD остаётся единственным FTP-движком;
- PASV всегда возвращает `192.168.0.179`;
- TCP 21, PASV 40000-40100;
- обычный FTP разрешён;
- explicit TLS 1.2 остаётся доступен как опция;
- удалена LAN/WAN/NAT-логика `mod_ifsession`;
- существующие FTP-аккаунты и пароли сохраняются;
- пароль admin не изменяется;
- Nginx, SQL, сайты и боты не затрагиваются.
