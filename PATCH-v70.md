# HYPER-HOST v70

Исправляет ложную проверку v69 и восстанавливает отсутствующие vhost сайтов.

- Проверка больше не использует 127.0.0.1 одним запросом.
- Используется `curl --resolve DOMAIN:80:192.168.0.179` с ожиданием graceful reload.
- `add-site` после reload проверяет реальный probe-файл из public_html.
- Переданный домен принудительно получает/восстанавливает vhost без удаления файлов.
- FTP/FTPS, SQL, боты и пароль admin не меняются.
