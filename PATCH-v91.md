# HYPER-HOST v91 — фирменный установщик и меню

## Что изменено

- Добавлен отдельный файл `setup.sh` с интерактивным меню.
- Добавлен крупный фирменный заголовок **HYPER-HOST**.
- Во всех сообщениях установщика бренд **HYPER-HOST** выделяется цветом и жирным текстом.
- В меню доступны установка/обновление, ремонт, проверка Nginx, выпуск SSL и информация о проекте.
- После установки меню доступно командой `sudo hyper-host-installer`.
- Финальный экран показывает URL панели, логин, пароль, каталоги, команды, сайт, GitHub и разработчика.
- Сохранены исправления SSL/ACME/Certbot из v89 и v90.

## Запуск

```bash
chmod +x setup.sh install.sh
sudo bash setup.sh
```
