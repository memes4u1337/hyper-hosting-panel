# HYPER-HOST v73

- Исправлен скрытый probe-файл панели, блокировавшийся правилом `location ~ /\.`.
- Проверка панели использует обычный файл без ведущей точки.
- Повторно восстанавливаются site/panel vhost и существующие SSL-сертификаты по SAN/CN.
- Проверяется document root панели, главная страница и каждый HTTPS-сертификат через SNI.
- При реальной ошибке откатывается только Nginx runtime и CLI.
- FTP, SQL, боты, файлы сайтов и пароль admin не изменяются.
