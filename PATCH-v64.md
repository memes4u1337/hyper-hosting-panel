# HYPER-HOST v64

Точечное исправление установщика ProFTPD/FTPS.

## Исправлено

- файл назначения для `lftp get` больше не создаётся заранее через `mktemp`;
- используется отдельный временный каталог;
- включён `xfer:clobber true`;
- после скачивания проверяется наличие файла и совпадение SHA-256;
- сохранён автоматический откат при реальной ошибке FTPS;
- Nginx, SQL, сайты и пароль `admin` не меняются.
